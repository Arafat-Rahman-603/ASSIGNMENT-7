# KeenKeeper

A personal friendship management dashboard designed to help you browse, tend, and nurture the relationships that matter most. Built for the modern web with an emphasis on a clean, premium user experience.

## Features

- **Dashboard Overview**: Get a bird's-eye view of your relationships, tracking who needs attention and who you're on track with.
- **Friend Detail Pages**: Deep dive into individual connections. Track your relationship goals and log interactions.
- **Interaction Logging**: Quickly log calls, texts, and video chats with toast notifications to confirm your check-ins.
- **Timeline History**: Review a complete history of all your logged interactions, complete with filtering options by interaction type.
- **Analytics**: Visualize your communication habits with an interactive pie chart showing the breakdown of your interaction types.

## Technologies Used

- **Next.js (App Router)**: For the core React framework, routing, and fast performance.
- **Tailwind CSS**: For all styling, ensuring a fully responsive and pixel-perfect design matching the Figma specs.
- **Recharts**: For the Friendship Analytics pie chart visualization.
- **Lucide-React**: For beautiful, consistent iconography across the application.
- **React Hot Toast**: For smooth, accessible notification popups when logging interactions.
- **Context API & LocalStorage**: For client-side state management and data persistence without needing a backend.
